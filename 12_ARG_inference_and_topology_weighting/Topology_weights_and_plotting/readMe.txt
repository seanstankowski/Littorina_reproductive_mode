This file contains all of the code needed to produce the Tr weights from the ARG shown in Fig. 3E.

.trees:
These file contain the sequences of tree in extended newick format, that were produced by ARGweaver and passed to Twisst for topology weighting. 

.weights_trees:
These files contain the topology weights for each tree, as generated by twisst

plotting.txt:
This file contains the code needed to plot their weights across the genome, as shown in Fig. 3E.
