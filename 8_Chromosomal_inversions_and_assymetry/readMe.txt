This file contains all of the code used to analyse toplogy weights for he known chromosomal inversions in Littorina. This includes performing a PCA, creating plots, and analyzing symmetry in the ternary framework.

This jupyter notebook contains all of the analysis code needed to produce the result and figures associated with fig. S20, S21 and S22.

Genotyping_and_PCA_for_LGC2.1.R:
This file contains the code used to conduct the PCA for inversion LGC2.1, as shown in fig. S23.

The other files in the director are used and described in Understanding_Tc_bias.ipynb.
